# 📱 FisioApp - Frontend (PWA)

Aplicação web progressiva (PWA) desenvolvida como parte do desafio da disciplina de Coding: Mobile.  
O sistema permite o cadastramento e gerenciamento de procedimentos de fisioterapia.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Service Workers (PWA)
- Manifest.json

---

## 📂 Estrutura do Projeto

```bash
physio-control-frontend/
│
├── index.html          # Estrutura principal da aplicação
├── service-worker.js   # Service Worker (PWA / offline)
├── manifest.json       # Configurações do PWA
├── README.md           # Documentação do projeto
├── style.css       # Estilização da aplicação
├── app.js       # Lógica da aplicação (API)
├── icons/
│   ├── icon-192.png    # Ícone PWA (192x192)
│   └── icon-512.png    # Ícone PWA (512x512)
```
---

## ⚙️ Funcionalidades

- Cadastro de procedimentos de Fisioterapêuticos
- Listagem de procedimentos
- Atualização (CRUD)
- Exclusão de registros
- Funciona offline (PWA)
- Instalável como aplicativo
